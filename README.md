# Royale One (monorepo)

Public brand: **Royale One**. This repository folder may still be named `barcelona-city-os` on disk; product and package metadata use the Royale One naming strategy.

## Apps

- `apps/web`: Next.js 16 web app with locale routing, premium landing page, theme toggle, and auth screens
- `apps/dispatcher`: Go dispatcher scaffold with a health endpoint

## Packages

- `packages/ui`: shared UI primitives
- `packages/types`: shared Zod schemas and TypeScript types
- `packages/database`: database scaffold for future Drizzle/PostGIS wiring
